# Exploring-Visualising-Data

Skills: Exploratory Data Analysis (EDA), visualisation, statistical methods.
Dataset: World Bank World Development Indicators (CSV)
https://www.kaggle.com/datasets/kaggle/world-development-indicators

#### Tasks:

1. Conduct initial data investigation: check dataset size, missing values, and summary stats.

2. Restructure the dataset (pivot tables, filtering, grouping).

3. Check data integrity: detect anomalies, duplicates, and inconsistencies.

4. Find significant patterns and trends

5. Apply statistical methods: mean, correlation, hypothesis testing.

6. Create basic visualisations (scatter plots, bar charts, line graphs).

#### Deliverable: EDA notebook and a short Exploration Report with plots and key insights.
